## 三、功能模块（官网功能区块）

### 01 · 拆分 PDF

将一份多页 PDF 按页拆成多个单页文件。适合归档、分发单页材料。

### 02 · 合并 PDF

把多个 PDF 合成一份。支持拖拽调整合并顺序，先加的文件不一定排在最前。

### 03 · 旋转页面

整份 PDF 统一旋转 90°、180° 或 270°。选择角度后，预览区立即显示效果。

### 04 · 格式转换

- **转入 PDF**：图片、纯文本、常见 Office 文档
- **转出图片**：PDF 导出为 PNG 或 JPEG

### 05 · 压缩 PDF

三档压缩强度，在体积与清晰度之间按需选择，适合邮件附件与网盘分享。

### 06 · 文字水印

为 PDF 每一页添加自定义文字水印。可选 1–5 条对角线平铺，预览与导出效果一致；最近使用的水印文字会自动记住。

## 四、体验亮点（差异化表述）

**苹果风界面，紫色品牌点缀**  
白色卡片、柔和阴影、圆角控件；品牌色 `#8A5CF5`，浅色深色模式均适配。

**拖放即导入**  
文件拖入左侧区域即可添加；合并模式下可拖拽排序。

**看得见再处理**  
右侧滚动预览：PDF 多页、图片原图、Office 文档文字摘要，处理前心里更有数。

**处理有反馈**  
顶部进度条、完成 Toast 提示；自动记住上次输出目录，少点几次「另存为」。

**内置更新**  
设置菜单可检查新版本、查看更新日志、下载安装包。

## 八、常见问题 FAQ

**Q：和 Adobe / 福昕比有什么优势？**  
A：不做全能套件，只做日常最高频的六类操作，启动快、界面轻、无订阅。

**Q：Office 预览为什么是文字而不是原排版？**  
A：v0.1.0 采用文字提取预览，便于确认内容；排版还原不在当前版本范围。

**Q：支持 Mac 吗？**  
A：当前发布目标为 Windows；技术栈为 Qt，后续可评估跨平台构建。

**Q：水印预览和导出不一致怎么办？**  
A：v0.1.0 已统一 `watermarklayout` 算法，预览与导出共用同一套布局；若仍有问题请通过 Issues 反馈。

**Q：旧版 .doc / .ppt 能转 PDF 吗？**  
A：建议另存为 .docx / .pptx 后再导入；二进制旧格式暂不支持。

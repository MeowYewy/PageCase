PDF Studio v0.1.0 — Portable (Windows 64-bit)
==============================================

1. Extract this folder anywhere (e.g. Desktop\PDFStudio).
2. Double-click ProjectP.exe to run.
3. No installation required. Uninstall = delete the folder.

System requirements
-------------------
- Windows 10 / 11, 64-bit

Notes
-----
- First launch may trigger Windows SmartScreen (unsigned app). Choose "More info" → "Run anyway".
- PDF processing stays on your PC; files are not uploaded.
- Output folder is remembered between sessions.

Included tools (bundled)
------------------------
- qpdf — merge, split, rotate, compress, watermark
- poppler pdftoppm — PDF page preview (when Qt Pdf module is not bundled)
